**Clinical Metrics Grid Specification from SPEC.md Section 7:**
- 2-column LazyVGrid, 10pt spacing
- Card: RoundedRectangle(cornerRadius: 18), secondarySystemBackground, 16pt padding

**Standard cards (4 cards in 2x2 grid):**
Each card contains:
- SF Symbol icon: 20pt, secondaryLabel color, .title3
- Label: .caption2, uppercase, tertiary
- Primary value: .title3, bold, primary
- Unit: .subheadline, secondary, inline with value
- Sub-label: .caption, tertiary

| Card | SF Symbol | Label | Value Source | Notes |
|------|-----------|-------|--------------|-------|
| SpO₂ en episodio | lungs.fill | SpO₂ EN EPISODIO | HKQuantityTypeIdentifier.oxygenSaturation × episode window | Show "No data" if no coincident reading |
| HRV (SR-filtrado) | waveform.path.ecg | HRV (EN RS) | heartRateVariabilitySDNN × SR windows | Label "est." in exports |
| Resp. ventricular AF | bolt.heart.fill | RESP. VENTRICULAR AF | heartRate × AF windows | Always show "est." suffix |
| Duración episodio | clock.fill | DURACIÓN EP. | Derived from event intervals | Duration + onset time |

**Wide card (full-width):**
- Label: "SÍNTOMAS SIN RITMO EXPLICATIVO"
- Value: Integer count, .title3, bold
- Sub-label: Recent unmatched capture timestamp + "Ver correlación →" button

**SpO₂ display rule:**
- Only display when reading exists within ±15 min of episode
- Otherwise show "No data" + sub-label "No datos durante el episodio"
- Never display SpO₂ from outside episode window
</context>
