**Rhythm Map Specification from SPEC.md Section 6:**
- Card geometry: Same as Burden Card (20pt corner radius, secondarySystemBackground)
- 24 bars, equal width, 2pt gap
- Bar: RoundedRectangle(cornerRadius: 3) with top corners only
- Max height: 36pt, bottom-aligned
- Bar colors:
  - Sinus rhythm: systemBlue at 50% opacity
  - Atrial fibrillation: RhythmAF at 90% opacity  
  - No data: systemFill, minimum 4pt height
- Bar height: Encodes normalized relative heart rate
- Sample count rule: < 3 samples = 40% opacity + "Datos insuficientes" warning

**Time axis:** Labels at 00:00, 06:00, 12:00, 18:00, 23:00 - .caption2, quaternary

**Legend:** Divider then three labels: "Ritmo sinusal", "Fibrilación auricular", "Sin datos"
- Legend squares: 8x8 RoundedRectangle(cornerRadius: 2)

**Interaction:**
- Tap card → Navigate to full Rhythm Map detail view
- Tap bar → Popover/tooltip with hour range, classification, sample count
- If sample count < 3: Show warning "Datos insuficientes — clasificación estimada"
