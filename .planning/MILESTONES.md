# Milestones

## v0.3 UI Improvements (Shipped: 2026-03-19)

**Phases completed:** 6 phases, 8 plans

**Key accomplishments:**
- Multi-window AF burden analysis with clinical thresholds (ASSERT/TRESTS)
- Symptom-rhythm correlation with ±30min window detection
